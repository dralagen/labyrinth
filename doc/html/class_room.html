Affiche le contenue de la salle.

Réalise un affichage sur 3 lignes en fonction de son état (visited, unvisited, position)
visited : Affiche le contenue de la salle
unvisited : Affiche un brouillard (masque le contenue)
position : Indique ma position dans le labyrinthe
